📘 StudyFlow-AI
Context-Aware Collaborative Learning Platform
🚀 Overview

StudyFlow-AI is an intelligent learning platform that helps students convert unstructured notes into meaningful knowledge, generate adaptive quizzes, and collaborate with peers for effective studying.

It combines AI + note-taking + revision + collaboration into one seamless system.

✨ Features
📝 AI-Powered Note Structuring
Convert raw notes into:
Overview
Key Concepts
Core Principles
Formulas
Summary
Improves understanding and retention
🧠 Smart Quiz Generator
Auto-generates quizzes from notes
Adjustable:
Difficulty levels
Number of questions
Tracks performance and accuracy
👥 Peer Matching System
Find study partners based on:
Subject
Learning style
Compatibility-based matching
Real-time availability
💬 Collaborative Study Sessions
Group chat for discussions
AI assistant support inside chat (@ai)
Helps in doubt solving and explanations
📊 Additional Features
Clean and modern UI
Structured vs raw note toggle
Concept-based learning assistance
Progress tracking
🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript
Design: Custom CSS (Dark UI)
AI Integration: Anthropic Claude API (optional)
Architecture: Single Page Application (SPA)
